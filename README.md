# Parallel Computing and Distributed Systems

This repository contains laboratory works completed by **Iryna Sabat** during the 3rd year of studies at **Lviv Polytechnic National University**.

## 📂 Project Structure
* **Lab1-ConvexHull**: Parallel implementation of the Monotone Chain algorithm to find the convex hull and determine linear separability.

## 🛠 Technologies
* **Language:** Java 17+
* **Parallelism:** ForkJoinPool, Parallel Streams, CompletableFuture.
* **Build System:** IntelliJ IDEA Project.

## 🚀 How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Navigate to the specific Lab folder and run `Main.java`.
